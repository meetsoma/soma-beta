# v0.28.1 → v0.32.0 (consolidated arc)

## Actions

### Settings: `systemPrompt.includeSystemCore` (new, default true)

SX-780 added the toggle that gates the universal harness layer (`prompts/system-core.md` — breath,
memory, preloads, how-to-work) into the system prompt. Default **on** (dogfood + new users get the
harness). Power users whose body files already cover the harness can set it `false` to skip the
redundancy. Absent key = on (code default), so existing installs are unaffected; the sentinel just
makes it explicit.

### Template: `_memory.md` — new `## The Through-Line` section

The bundled preload template gained `## The Through-Line` — the single most valuable transfer (the
mental model / reframe to inherit, not state). Cross-soma convergence (gravicity-studio + arzadon-handoff
both evolved it independently). Additive HTML-comment guidance after `## Resume Point`. Pristine copies
auto-update; customized `_memory.md` is flagged by the doctor (your edits preserved).

## What changed (highlights)

### Headless sub-agents on Claude via the official CLI (SX-782)

`soma:agent.delegate({ model: 'claude-cli/sonnet' })` + `soma claude "<task>"` run a child through the
OFFICIAL `claude -p` — **first-party = subscription plan, not extra-usage** (the only headless-Claude
path that doesn't bill overage). Read-only tools by default; role opts into write/bash.

### system-core.md was silently dropped from every prompt — fixed (SX-780)

A path bug made the whole harness layer (breath/memory/preloads/how-to-work) absent from every compiled
prompt for months. Fixed the load path + added the `includeSystemCore` toggle + a prompt-content smoke
that asserts every harness section renders.

### The children pattern is now discoverable (SX-784)

`soma:agent.delegate` help carries `## The children pattern` (author durable roles in
`body/children/<role>.md`; the sub-compiler injects each role's `## Accumulated Knowledge` every spawn).
Plus a docs guide on authoring children.

### Statusline fixes (SX-783, SX-781)

Preload-saved notice now persists (mtime-gated reconcile); the runtime-branch indicator always renders
(dim `(main)` on plain main); the keepalive timer no longer crashes on a session-replacement stale ctx.

### Release hardening (SX-777, SX-785)

Post-ship narrative reconcile (`--narrative-final`); a fail-fast migration gate (Phase 0.6) that blocks
a cut when settings/templates changed without a migration file; a pre-push credential-file tree-scan.
